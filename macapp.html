为APP制作图标

创建icons.iconset文件夹

mkdir icons.iconset

生成图片集

sips -z 16 16 ali.png --out icons.iconset/icon_16x16.png
sips -z 32 32 ali.png --out icons.iconset/icon_16x16@2x.png
sips -z 32 32 ali.png --out icons.iconset/icon_32x32.png
sips -z 64 64 ali.png --out icons.iconset/icon_32x32@2x.png
sips -z 128 128 ali.png --out icons.iconset/icon_128x128.png
sips -z 256 256 ali.png --out icons.iconset/icon_128x128@2x.png
sips -z 256 256 ali.png --out icons.iconset/icon_256x256.png
sips -z 512 512 ali.png --out icons.iconset/icon_256x256@2x.png
sips -z 512 512 ali.png --out icons.iconset/icon_512x512.png
sips -z 1024 1024 ali.png --out icons.iconset/icon_512x512@2x.png

转化成icon.icns

iconutil -c icns -o icon.icns icons.iconset

制作 .app bundle

mac上运行的程序其实上一个.app的文件夹，其中包含了二进制文件和清单文件。

目录结构如下；

$ tree my.app
my.app/
└── Contents
    ├── Info.plist
    ├── MacOS
    │   └── my
    └── Resources
        └── icon.icns
3 directories, 3 files

其中有三个关键的文件：

  • Info.plist:清单文件
  • my: 你要打包的二进制文件
  • icon.icns: 打包好的不同尺寸的图标文件

添加Info.plist

文件示例

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>CFBundleExecutable</key>
    <string>my</string>
    <key>CFBundleIconFile</key>
    <string>icon.icns</string>
    <key>CFBundleIdentifier</key>
    <string>com.example.yours</string>
    <key>NSHighResolutionCapable</key>
    <true/>
    <key>LSUIElement</key>
    <true/>
</dict>
</plist>

其中你需要更改的地方为CFBundleExecutabl、CFBundleIdentifier

ps：LSUIElement 告诉macos这个程序是一个代理程序，以至于它不会出现在任务栏中。起到了一定的隐藏效果。。

现在你就得到了一个可以双击打开的app了

制作DMG文件

制作一个磁盘文件，其中放置了你的app，这样可以轻易的拖拽到应用文件夹并且安装。

制作 DMB 模板

打开磁盘工具。按 ⌘N 创建一个新的磁盘镜像。给它取个名字，配置好足以容纳你的程序包的空间大小。
